===============================================================

Процесс выполнения задачи

Пользователь создает задачу.
   Перед созданием задачи пользователь указывает:
      - Название
      - Описание
      - Список подчиненных задач
      - Желаемые дату/время завершения задачи
      - Текущее состояние задачи:
         - Исполнение
         - Приостановлено

   При создании задачи система указывает:
      - Дату/Время создания задачи
      - Поля поумолчанию:
         - Общее время выполнения задачи = 0
         - Текущее состояние задачи = приостановлено
      - Вносит указанные пользователем перед созданием задачи поля
      - Указывает дату/время изм-я задачи

Пользователь редактирует задачу
   Перед сохранением пользователь указывает:
      - Все поля
      - Меняет поле "Текущее состояние задачи"
   При сохранении указывается
      - дата и время изменения задачи
      - При смене статуса на "Исполнение":
         Запоминается дата/время установки этого статуса(время последней установки данного статуса)
      - При смене статуса на "Приостановлено":
         Расчитывается общее время исполнения задачи и записвается в специальное поле.


Вычисление времени исполнения задачи:
   Если статус задачи "Исполнение", то за крайнее время берется текущий момент.
   Если статус задачи "Приостановлено", то крайнее время - это время изменения задачи
   Разность текущего времени и крайнего времени
